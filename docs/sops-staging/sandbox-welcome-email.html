The NCC Sandbox Is Live

Explore every feature. Break nothing. Learn everything.

We've launched a free sandbox environment inside NCC where you can explore the full platform — estimating, project management, scheduling, financials, documents, and more — without affecting any live data.

Everyone gets full Admin access automatically. Just log in and switch to "NCC Sandbox" from the company switcher.

Log In & Try the Sandbox

🏗️ For General Contractors & Principals

See how NCC manages your entire operation before committing a single dollar. The sandbox gives you a fully functional environment to evaluate:

  • Estimating & Xactimate integration — import price lists, build estimates, track cost variance
  • Project financials — invoicing, payment applications, draw schedules, P&L by project
  • Subcontractor management — bid packages, sub agreements, compliance tracking
  • Workforce scheduling — crew allocation, daily logs, timecard automation
  • Document control — contracts, change orders, RFIs, submittals — all version-controlled

This is the same platform your team would use in production. Nothing is simulated.

👷 For Project Managers, Estimators & Field Teams

Get hands-on with the tools you'll use every day — no training videos, just real workflows:

  • Create test projects — build estimates, assign crews, run the full project lifecycle
  • Try daily logs & timecards — see how field reporting flows from site to payroll
  • Explore the document system — upload, tag, version, and share files
  • Run financial reports — project P&L, WIP, cost-to-complete, and cash flow
  • Test messaging & notifications — see how the communication layer works across teams

Everything you create stays in the sandbox. Your real company data is completely separate.

⚙️ For Administrators

Evaluate the admin experience and configuration options before rolling NCC out to your organization:

  • Company settings — branding, office locations, default configurations
  • Module exploration — enable/disable features and see how they integrate
  • Reporting dashboards — financial summaries, project health, workforce metrics
  • Template management — agreements, documents, checklists, and workflows
  • Integration preview — see connection points for Xactimate, accounting, payroll

Note: A few destructive actions (deleting projects, changing user roles, deactivating members) are disabled in the sandbox to keep it stable for everyone.

How It Works

  1. Log in to NCC at staging-ncc.nfsgrp.com
  2. Open the company switcher (top-left dropdown)
  3. Select "NCC Sandbox"
  4. Explore freely — you have full Admin access

Inactive for 30 days? Your sandbox access pauses automatically. Just switch back anytime to reactivate.
