## Purpose

Manage tasks with color-coded urgency (red=overdue, yellow=today, green=ahead) and badge counts. Tasks from daily log observations auto-link back to the originating log.

## Powered By — CAM Reference

> **OPS-VIS-0002 — Urgency-Based Task Dashboard** (29/40 ✅ Qualified)
> *Why this matters:* Tasks and daily logs are separate silos in every other tool. NCC links them — a field observation creates a task that links back to the log entry. The urgency dashboard with 60-second refresh means PMs always know what's overdue.
